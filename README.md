## Shawnee Mission Park Culvert Inventory 
### Preliminary Map of Culverts
<img width="2000" height="1294" alt="image" src="https://github.com/user-attachments/assets/531ae703-5196-42aa-bd2b-d7ee5849449d" />
The main project for this internship was to make a digital culvert inventory of JCPRD properties, and the first park we collected data from was Shawnee Mission. The only inventory that JCPRD had was a remote sensing image with vague locations for some culverts. This information had not been updated in over 10 years, so many new culverts were missing, and the locations were not precise enough for maintenance workers.
